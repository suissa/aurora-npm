# Aurora पैकेज मैनेजर

**Aurora Austral** भाषा के लिए एक उच्च-प्रदर्शन, स्टैंडअलोन पैकेज मैनेजर, जिसे बिना किसी **बाहरी निर्भरता** के डिज़ाइन किया गया है।

## इंस्टॉलेशन

```bash
npm install -g @aurora.purecore.codes/latest@1.1.0
```

**नोट:** यह पैकेज पूरी तरह से स्वतंत्र है। यह केवल देशी Node.js API (`fetch`, `fs`, `path`, `os`, `child_process`) का उपयोग करता है।

## मुख्य कमांड

- `aurora init`: एक नया प्रोजेक्ट शुरू करता है और स्थानीय कंपाइलर सेट करता है।
- `aurora install <package>`: आधिकारिक रिपॉजिटरी से पैकेज डाउनलोड, कंपाइल और टेस्ट करता है।
- `aurora find <query>`: नाम और README.md फ़ाइलों के भीतर बुद्धिमान खोज।
- `aurora list`: उपलब्ध या स्थानीय रूप से इंस्टॉल किए गए पैकेज की सूची दिखाता है।
- `aurora update`: प्रोजेक्ट की निर्भरता को अपडेट करता है।

## मानक लाइब्रेरी एकीकरण
मैनेजर स्वचालित रूप से `aurora-austral-standard-lib` के स्थान का पता लगाता है ताकि यह सुनिश्चित हो सके कि सभी पैकेज मानक लाइब्रेरी के स्थानीय संस्करण के खिलाफ सही ढंग से कंपाइल किए गए हैं।
