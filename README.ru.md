# Менеджер пакетов Aurora

Высокопроизводительный автономный менеджер пакетов для языка **Aurora Austral**, разработанный без **внешних зависимостей**.

## Установка

```bash
npm install -g @aurora.purecore.codes/latest@1.1.0
```

**Примечание:** Этот пакет полностью автономен. Он использует только встроенные API Node.js (`fetch`, `fs`, `path`, `os`, `child_process`).

## Основные команды

- `aurora init`: Инициализирует новый проект и настраивает локальный компилятор.
- `aurora install <пакет>`: Скачивает, компилирует и тестирует пакет из официального репозитория.
- `aurora find <запрос>`: Интеллектуальный поиск по именам и внутри файлов README.md.
- `aurora list`: Список доступных или установленных локально пакетов.
- `aurora update`: Обновляет зависимости проекта.

## Интеграция со стандартной библиотекой
Менеджер автоматически определяет местоположение `aurora-austral-standard-lib`, чтобы гарантировать правильную компиляцию всех пакетов с использованием локальной версии стандартной библиотеки.
